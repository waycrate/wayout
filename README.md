# Usage:
```bash
wayout -o <your output name> -s <on/off/toggle>
```
# Installation

## AUR:
`wayout-git` will be packaged soon.

## Compile time dependencies:
-   rustup
-   make

## Compiling:
-   `git clone https://github.com/waycrate/wayout && cd wayout`
-   `make setup`
-   `make`
-   `sudo make install`
